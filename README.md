# CIBOserver

This is a Machine Learning recipe recommender system which recommends dishes based on the ingredients provided by the user

![image](https://github.com/AbhishekGusain1289/CIBOserver/blob/main/images/dish-final.png)

You can select the ingredients are required and the will recommend you dishes that you can make.
