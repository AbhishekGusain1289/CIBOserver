# CIBOserver

A Machine Learning-powered recipe recommender system that suggests delicious dishes based on the ingredients you have available.

## Features

- **Smart Recommendations**: Get dish suggestions based on your available ingredients
- **Ingredient Selection**: Easily select multiple ingredients from an intuitive interface
- **Recipe Finder**: Discover recipes you can actually make with what you have

## How It Works

Simply select the ingredients you have on hand, and the system will recommend dishes that you can prepare.
